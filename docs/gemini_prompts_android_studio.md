
# Gemini in Android Studio プロンプト集（個人開発向け）

> Android Studio 内でそのまま使える、実戦テンプレートを厳選しました。

## 0) 万能テンプレ（まずはこれ）
```text
あなたはAndroid/Kotlin/Jetpack Composeの上級エンジニアです。
目的：{やりたいこと / 解決したい問題}
前提：{アプリ構成・該当画面・使用ライブラリ}
制約：{最小変更 / 互換性 / 既存仕様を壊さない / 例: minSdk / Compose version}
入力：{エラーログ or 該当コード}
求める出力：
1) 原因の候補（優先順位つきで3つ）
2) 切り分け手順（確認項目・再現手順）
3) 修正案（差分コード）+ 代替案
4) 影響範囲（テスト観点）
形式：見出し付き。差分はコードブロックで。
```

## 8) 無料枠で精度を上げる小技
```text
- 入力は「最小再現」に絞る（例外の先頭〜Caused by〜自前コード行）。
- 出力形式を縛る（原因→切り分け→差分コード→影響範囲）。
- 1回で完結させず、仮説→確定→差分の2段階で詰める。
- 機密・不要ファイルは .aiexclude で除外する。
```
