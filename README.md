# 社会福祉士 毎日3問トレーナー

「社会福祉士 毎日3問トレーナー」は、社会福祉士国家試験の合格を目指す学習者を支援するためのAndroidアプリです。毎日少しずつクイズ形式で問題を解くことで、継続的な学習をサポートします。

## 主な特徴
- **毎日の継続学習**: 1日1セット3問のクイズで、無理なく学習を継続できます。
- **5分野の学習分析**: 「人・社会・福祉」「権利・制度・基盤」「相談援助の理論と方法」「各対象者向けサービス」「総合問題」の5分野で得意・不得意を可視化。
- **プレミアム機能**: 広告非表示、詳細な解説の全表示、学習履歴の無制限閲覧などが可能。
